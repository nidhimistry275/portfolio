# Portfolio Website

## Overview
This is the personal portfolio website, hosted on Netlify: [Portfolio Link](https://portfolionidhi.netlify.app/). It showcases my projects, skills, and experience in software development.

## Tech Stack
- **Frontend:** ReactJS
- **Styling:** CSS, Bootstrap
- **Hosting:** Netlify

## Features
- Fully responsive design for seamless viewing on all devices
- Showcases my projects and professional experience
- Includes a contact section for inquiries and collaborations

## Deployment
The website is deployed using Netlify, ensuring continuous deployment with each update to the repository.
